K8S CONTROL PLANE

Mixed Version Proxy graduates to Beta in v1.36

The Mixed Version Proxy now routes API requests to a peer apiserver that knows the resource version, avoiding spurious 404s during upgrades. Enabled by default in v1.36 — relevant for any fleet doing rolling control-plane upgrades.

kubernetes.io Reduces a class of upgrade-time false negatives that incident-response agents must learn to ignore.

AGENT REASONING

AI plus formal verification

Kleppmann argues that LLMs paired with formal verifiers can produce code and actions with checkable guarantees, not just plausible outputs. A useful frame for designing evidence gates and mitigation boundaries in autonomous remediation.

martin.kleppmann.com Sharpens the safety argument for autonomous-remediation research without overclaiming.

OPERATING MEMORY

Memory is state, not a service

Essay argues durable agent memory should be modeled as reviewable state with explicit reads and writes, not a black-box retrieval service. Directly applicable to building an engineering knowledge cockpit on top of a second brain.

nanothoughts.substack.com Anchors operating-memory design choices around auditability and replay.

LLM SERVING

vLLM v0.21.0 — HMA, KV offload, speculative thinking budgets

vLLM 0.21.0 integrates KV offload with the Hybrid Memory Allocator and adds speculative decoding that respects a thinking budget. Build now requires C++20 and transformers v4 is deprecated.

github.com/vllm-project Affects routing assumptions and memory headroom for any inference platform tracking vLLM.

INCIDENT TRIAGE

KV eviction needs structural protection

Paper shows seven KV-eviction policies collapse to near-zero quality without structural protection at prompt boundaries; reserving 10% recovers most of the reference-ceiling quality. A concrete failure mode worth carrying into reliability reasoning for LLM-backed agents.

arxiv.org Concrete example of a silent-degradation pattern that an incident agent should learn to detect.

GATEWAY API

Inference Extension v1.5.0 lands latency predictor and response-path mirroring

Gateway API Inference Extension v1.5.0 ships a sliding-window latency predictor training server and mirrors request plugins onto the response path. Opportunity to read the new plugin contracts before adopting AI-aware routing.

github.com/kubernetes-sigs Useful reference point for evaluating model-aware routing without committing to a stack.
